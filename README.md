# Jogo Adivinhe o Número

Um jogo simples onde o usuário tenta adivinhar um número aleatório gerado pelo computador.

---

## Tecnologias Utilizadas
- **HTML**: Estrutura da página e elementos do jogo.
- **CSS**: Estilização e layout visual.
- **JavaScript**: Lógica do jogo e interação com o usuário.

---

## Como Jogar
1. Abra o arquivo `index.html` no navegador.
2. Tente adivinhar o número que o computador escolheu.
3. Digite sua aposta no campo indicado e clique no botão para verificar.
4. O jogo irá informar se o número é maior, menor ou se você acertou.
5. Continue tentando até acertar o número secreto.

---

## Funcionalidades
- Geração aleatória de número secreto.
- Feedback ao usuário se o palpite está alto ou baixo.
- Contador de tentativas.
- Interface interativa e amigável.
